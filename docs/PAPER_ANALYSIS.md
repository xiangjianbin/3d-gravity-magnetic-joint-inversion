## 1. 方法论核心

### 1.1 整体网络架构（5 任务）

| Task | 名称 | 输入 | 输出 | 损失函数 |
|------|------|------|------|----------|
| Task 1 | 独立重力反演 | Backbone特征 | 密度模型 M_ρ_pred | MSE |
| Task 2 | 独立磁法反演 | Backbone特征 | 磁化率模型 M_κ_pred | MSE |
| Task 3 | 结构相似性提取 | M_ρ_pred + M_κ_pred | 结构相似图 S | MSE |
| Task 4 | 联合重力反演 | 原始输入 + S | 最终密度 M_ρ_final | BCE (ω=1) |
| Task 5 | 联合磁法反演 | 原始输入 + S | 最终磁化率 M_κ_final | BCE (ω=1) |

**前向传播顺序**:
```
Input [B, 2, 40, 40, 20] → Backbone U-Net → ASPP → Task1, Task2
Task1输出 + Task2输出 → Task3 → S
[Input, S] → Task4 → M_ρ_final (推理时使用)
[Input, S] → Task5 → M_κ_final (推理时使用)
```

### 1.2 Backbone 子网络（Fig.2）

- **输入尺寸**: 40 × 40 × 20 (Easting × Northing × Depth)
- **编码器** (4层下采样):
  - L1: Conv3d(2→64, k=3, s=1, p=1) + LeakyReLU + MaxPool3d(2) → [64,20,20,10]
  - L2: Conv3d(64→128, k=3) + LeakyReLU + MaxPool3d(2) → [128,10,10,5]
  - L3: Conv3d(128→256, k=3) + LeakyReLU + MaxPool3d(2) → [256,5,5,2]
  - L4: Conv3d(256→512, k=3) + LeakyReLU + MaxPool3d(2) → [512,2,2,1]
  - Bottleneck: Conv3d(512→1024, k=3) + LeakyReLU → [1024,2,2,1]
- **解码器**: 对称上采样 + Skip Connection
- **激活函数**: LeakyReLU (Eq.8, gamma 参数)

### 1.3 ASPP 模块（Fig.4）

- **位置**: Backbone 输出之后，Task 头之前
- **膨胀率**: {6, 12, 18} 三个并行分支
- **每分支**: Conv3d(256, 256, k=3, dilation=d) + BN + ReLU
- **全局池化分支**: AdaptiveAvgPool3d(1) → Conv3d(256,256,k=1) → Upsample
- **融合**: Concat(5分支) → Conv3d(256*5, 256, k=1)

### 1.4 独立反演子网络（Task 1 & 2, Fig.3）

- **输入**: ASPP 输出特征 [B, 256, 40, 40, 20]
- **结构**: 4 层小型 U-Net 解码器
- **内部 Skip Connection**
- **输出**: 单通道 [B, 1, 40, 40, 20]

### 1.5 结构相似性模块（Task 3）

- **输入**: [M_ρ_pred, M_κ_pred] 各自独立处理
- **流程**: Conv3d(k=3) → MaxPool3d(2) × 2 → 径向网格操作 → Conv3d → MaxPool3d
- **输出**: Sigmoid 激活 → 二值化结构相似图 [B, 1, 40, 40, 20]
- **物理含义**: S≈1 表示密度和磁化率在该位置同时非零（结构一致）

### 1.6 联合反演子网络（Task 4 & 5）

- **输入**: 原始 2ch 数据 + 结构相似图 S → concat 为 3ch
- **结构**: 类似 Task 1/2 的解码器
- **损失**: BCE (Binary Cross-Entropy, ω=1)

## 3. 正演公式

### 3.1 重力正演

基于 Blakely (1995) 棱柱体重力公式:

$$g_z = G \cdot \rho \cdot \sum_{cells} V_{cell} \cdot K_g(x,y,z)$$

其中:
- G = 6.674×10⁻¹¹ m³/(kg·s²) 万有引力常数
- ρ = 密度差 (g/cm³)
- K_g = 核函数（取决于棱柱体几何参数和观测点位置）

### 3.2 磁法正演

总场异常 ΔT 公式（感应磁化模型）:

$$\Delta T = \kappa \cdot F \cdot \sum_{cells} V_{cell} \cdot K_m(x,y,z; I, D)$$

其中:
- κ = 磁化率 (SI)
- F = 地磁场总强度 (nT)
- I = 磁倾角
- D = 磁偏角
- K_m = 磁法核函数

## 7. 实现要点总结

### 必须实现的核心组件

1. **重力正演引擎** — Blakely 1995 棱柱体公式
2. **磁法正演引擎** — 总场异常 ΔT 公式
3. **合成数据生成器** — 6 类地质体 × 45000 样本
4. **3D U-Net Backbone** — 4 层编解码器 + skip
5. **ASPP 模块** — dilation {6,12,18}
6. **5 个任务头** — 独立反演×2 + 结构相似性 + 联合反演×2
7. **训练管道** — AMP + 多任务损失
8. **评估管道** — MSE/RMSE/MAE/Correlation

### 复现成功标准

- 定量指标偏差 ≤ ±10%
- 主要地质体位置正确
- 训练曲线趋势与 Fig.5 一致（下降→收敛）
